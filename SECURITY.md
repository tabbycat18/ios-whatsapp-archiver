# Security

This project handles local files that may contain highly sensitive personal data. Treat all extracted archives, databases, media, and generated exports as private.

## Supported Use

- Run tools only on backups and archives you are authorized to access.
- Keep extracted archives outside version control.
- Review staged files before every commit.

## Reporting Issues

For security issues, avoid attaching private WhatsApp databases, screenshots, exports, or media files. Provide schema-only details, anonymized error messages, and reproduction steps using synthetic data whenever possible.
